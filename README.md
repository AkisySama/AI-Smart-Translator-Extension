# AI 词根翻译助手

一个 Chrome 浏览器扩展，选中网页上的英文单词或句子，即可获得 AI 驱动的翻译、词性和词根分析。

## ✨ 功能

- **单词分析** — 选中单词，获取中文释义、词性、词根及词根演变历史
- **句子翻译** — 选中短句，直接获得中文翻译
- **浮动触发图标** — 选中文本后出现图标，点击确认后才调用 API，避免误触发
- **智能过滤** — 自动忽略纯数字、中文文本、URL 等非英文内容
- **翻译缓存** — 内存级缓存，重复查询同一词汇秒出结果
- **弹窗自适应** — 弹窗自动调整位置，避免超出屏幕边界
- **多模型兼容** — 支持所有 OpenAI API 兼容接口（OpenAI、DeepSeek、通义千问等）

## 📸 使用方式

1. **安装扩展**：在 Chrome 中加载已解压的扩展目录
2. **配置 API**：点击工具栏扩展图标 → 填写 API URL、Key 和模型名称 → 保存
3. **使用**：在任意网页上选中英文单词或短句 → 点击浮出的翻译图标 → 查看结果

> ⚠️ 本插件需要自备 AI API Key，不内置任何翻译服务。

## 🔧 支持的语言

目前仅支持**英文**内容的翻译和分析。

## 🔒 隐私

- 所有 API 设置保存在浏览器本地存储中
- 选中的文本直接发送到你配置的 API 端点
- 插件本身不收集任何用户数据

## 🛠 技术栈

- Manifest V3
- Service Worker（后台）
- Content Script（页面注入）
- OpenAI 兼容 API
